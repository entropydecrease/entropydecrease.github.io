run job with at on Mac

entropydecrease published on 0001-01-01
Mac 上的 at 是由 atrun 运行的，而后者是由launchd 启动的，问题是在Mac上atrun默认是不启动的。 The atrun utility runs commands queued by at(1). It is invoked periodically by launchd(8) as specified in the com.apple.atrun.plist property list. By default the property list contains the Disabled key set to true, so atrun is never invoked. 以root身份运行如下命令以启动atrun: launchctl load -w /System/Library/LaunchDaemons/com.apple.atrun.plist 之后，就可以使用at命令来提交任务。
